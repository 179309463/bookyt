h1. CyBook

CyBook is a simple but powerfull financial accounting software.

CyBook is a web application implementing double-entry bookkeeping providing:

* Customizable accounts
* Booking wizard
* Booking templates
* Earnings report
* Balance sheet

CyBook is built on "Ruby On Rails":http://www.rubyonrails.org and released as "Open Source":http://www.opensource.org.
