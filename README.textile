h1. bookyt

bookyt is a simple but powerful financial accounting software.

h2. About bookyt

bookyt is a web application implementing double-entry bookkeeping providing:

* Customizable accounts
* Booking wizard
* Booking templates
* Earnings report
* Balance sheet
* Invoicing
* Accounts Receivable
* Accounts Payable
* Employee expense voucher
* Depreciation
* Setting up tax tables

bookyt is built on "Ruby On Rails":http://www.rubyonrails.org and released as "Open Source":http://www.opensource.org.
